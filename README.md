# Blinkit Sales Analysis
This project features an interactive, end-to-end data analysis dashboard built using Microsoft Power BI. Designed to provide a comprehensive view of sales performance for Blinkit (an online grocery delivery service), the dashboard enables data-driven decision-making through dynamic visualization.

## Tech Stack
The dashboard was built using the following tools and technologies:
* Power BI Desktop: Main data visualization platform used for report creation.
* Power Query: Data transformation and cleaning layer for reshaping and preparing the data.
* DAX (Data Analysis Expressions): Used for calculated measures, dynamic visuals, and conditional logic.
* Data Modeling: Relationships established to enable cross-filtering and aggregation.
* File Format: .pbix for development.

## Data Source
Source: Provided Blinkit grocery sales dataset (8,523 rows of transaction data).

The data contains granular information on product attributes (Fat Content, Item Type), outlet details (Outlet Size, Location Tier, Establishment Year), and performance metrics (Sales, Ratings).

## Features and Highlights

* **Business Problem**: With a large volume of daily transactions, the management team faces challenges in identifying which outlets perform best, which product categories drive the most revenue, and how to improve customer ratings.

* **Goal of the Dashboard**: To deliver an interactive visual tool that tracks KPIs, segments sales by product attributes, and provides regional/operational insights.

* **Walkthrough of Key Visuals**:
    * **Key KPIs (Top Section)**: Monitoring Total Sales (1.2M), Average Sales (141), Number of Items (8,523), and Average Ratings (3.9).
    * **Fat Content Analysis (Donut Chart)**: Visualizes the revenue split between Low Fat and Regular items.
    * **Item Type Distribution (Bar Chart)**: Ranks product categories by sales contribution.
    * **Outlet Establishment Trend (Line Chart)**: Tracks sales performance based on the year an outlet was established.
    * **Interactive Slicers**: A filter panel allows users to drill down into data by Outlet Location Tier, Outlet Size, and Item Type.

* **Business Impact**: Identifying top-selling item types allows for better stock allocation, comparing sales across outlet sizes helps in strategic planning, and monitoring average ratings helps identify areas for service improvement.
